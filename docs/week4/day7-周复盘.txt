一、四周成果一览
=================

▶ 项目代码（src/study_agent/）

  核心模块（7 个）：
    config/     — 5 家 LLM Provider 配置、LLMConfig、AppConfig、环境变量管理
    llm/        — LLMClient 统一客户端、重试机制（指数退避）、结构化输出提取
    prompt/     — Jinja2 模板引擎、Few-Shot 管理、评测框架、50 条测试用例
    tools/      — BaseTool / ToolCallLoop / Calculator / DateTime / TextStats
    rag/        — 分块(3种策略) / Embedding(3级降级) / 混合检索(RRF) / 生成与引用
    agent/      — PersonalQA Agent：整合以上全部，对话历史 + 多知识库管理
    api/        — FastAPI 接口层：/upload /chat /kb /conversations 共 10 个端点

  前端（frontend/）：
    Next.js + shadcn/ui 聊天界面
    — ChatArea / InputArea / Sidebar / MarkdownRenderer
    — Markdown 渲染 + 代码语法高亮
    — 知识库切换 + 对话历史管理
